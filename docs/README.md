# Territoire 2100

Plateforme d’aide à la décision climatique locale.

## Objectif
Montrer l’évolution historique du climat sur un territoire, projeter des scénarios futurs et proposer des actions concrètes adaptées aux citoyens et aux collectivités.

## MVP
- Région PACA
- Zoom sur Marseille / Bouches-du-Rhône
- 8 indicateurs
- Dashboard interactif
- Projections 2030 / 2050 / 2100
- Recommandations citoyennes

## Structure
- `docs/` : vision, décisions, backlog
- `data/` : données brutes et transformées
- `src/` : code source
- `notebooks/` : exploration et prototypage
- `tests/` : tests unitaires
