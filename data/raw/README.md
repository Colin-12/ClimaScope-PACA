# Raw Data

Ce dossier contient les données brutes téléchargées depuis les sources officielles.

## Règles
- ne jamais modifier un fichier brut manuellement
- toute transformation doit produire un nouveau fichier dans `data/processed/`
- tout téléchargement doit être tracé dans `data/raw/download_manifest.csv`
- éviter de committer de gros fichiers dans GitHub
